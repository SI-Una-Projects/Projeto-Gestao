# 📊 Sistema de Gestão de Projetos

## 📌 Descrição

Sistema desenvolvido para gerenciamento de projetos, equipes, usuários e tarefas.

## 🛠️ Tecnologias

* Java 21
* Spring Boot
* Spring Data JPA
* H2 Database

## 🧱 Arquitetura

Arquitetura em camadas baseada em MVC:

* Controller
* Service
* Repository
* Model
* DTO

## 👥 Equipe

* Nome 1
* Nome 2

## 🚀 Funcionalidades

* Cadastro de usuários
* Cadastro de projetos
* Cadastro de equipes
* Gerenciamento de tarefas
* Alocação de equipes em projetos
